Final

In this project I combined all three of my devices to create music that makes objects on a display. I wanted project to be used in a dark area because the trellis has really bright LEDs that can light up the dark. I want to combine that with my display where it creates shapes in random places making abstract shapes on the display and if done correctly it can become an animation. Furthermore, as the buttons are being pressed it creates tone, where someone can come to learn to make music with the different tones. This project was mainly an enclosure that makes bright color that really stand out in the dark while making objects and music.
